# Post-Processing Tests

This directory is reserved for test scripts and validation files.

## Purpose

- Test processors with sample data
- Validate export to OpenSearch/Horreum
- Integration testing

## Usage

Create test scripts here as needed for development and validation.

Examples:
- `test_<benchmark>_processor.py` - Test individual benchmark processors
- `test_opensearch.py` - Test OpenSearch export functionality
- `test_horreum.py` - Test Horreum export functionality

## Note

Test scripts are not committed to the repository. Add them to `.gitignore` if needed.
